The Go programming language is an open source project to make programmers more productive.

Go is expressive, concise, clean, and efficient. Its concurrency mechanisms make it easy to write programs that get the most out of multicore and networked machines, while its novel type system enables flexible and modular program construction.

Go compiles quickly to machine code yet has the convenience of garbage collection and the power of run-time reflection. It's a fast, statically typed, compiled language that feels like a dynamically typed, interpreted language.

The Go project includes the language itself, its tools and standard libraries, and last but not least, a cultural agenda of radical simplicity.

Go was originally designed at Google in 2007 to improve programming productivity in an era of multicore, networked machines and large codebases. Today it powers much of the cloud infrastructure across various companies and has become popular for CLI tools, DevOps, web development, and more.

The language is often referred to as Golang because of its domain name, golang.org, but its proper name is Go.
